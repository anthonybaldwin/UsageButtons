Open claude.ai in your browser, DevTools → Network → any request → copy the Cookie: header value. Only sessionKey= is required.

Windows + Chrome note: Chrome holds its Cookies database with exclusive access while running, so auto-import can't read it. To get the extra-usage buttons populated, quit Chrome once — the next refresh will scan, cache the cookie for 24 hours, and you can reopen Chrome immediately. Edge / Firefox / closed Chrome all work fine. Alternatively, switch this dropdown to Manual and paste the cookie header.
