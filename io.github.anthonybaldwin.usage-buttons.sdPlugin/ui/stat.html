Warn / critical thresholds

When the metric's value drops below these thresholds, the fill switches to the warn / critical color. For percent metrics the units are 0–100; for dollar metrics they're raw dollars. Leave blank to use per-metric defaults (20/10 for percent metrics, $10/$0 for dollar metrics — so a balance button goes orange below $10 and red when negative).
